
# HeartBeat Classification

This project implements a machine learning pipeline for **heartbeat sound classification**.  
It uses deep learning techniques to analyze audio recordings and classify them into different categories, supporting clinical research and diagnostic applications.

## 🔗 Project Links
- **GitHub Repository:** [HeartBeatClassification](https://github.com/ToobaRani01/HeartBeatClassification)  
- **Live Demo (Streamlit App):** [Heartbeat Classification App](https://toobaraniheartbeatclassification.streamlit.app/)

## 📌 Features
- Preprocessing of heartbeat audio signals
- Deep learning model for classification
- Interactive Streamlit interface for testing and visualization
- Easy deployment and reproducibility
